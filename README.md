# CS50's Introduction to Artificial Intelligence with Python

## Overview

This repository documents my completion of **CS50's Introduction to Artificial Intelligence with Python**, a course offered by HarvardX through edX. The course explores the fundamental concepts, algorithms, and techniques that power modern artificial intelligence systems.

Through a combination of theoretical learning and hands-on programming projects, I gained practical experience in designing and implementing intelligent systems using Python.
